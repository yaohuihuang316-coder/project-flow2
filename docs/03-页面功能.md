# 页面功能文档

## 前台页面

### Dashboard 仪表盘
- 学习进度概览
- 知识图谱快捷入口
- 推荐课程
- 最近活动

### LearningHub 学习中心
- 课程分类浏览
- 课程详情
- 学习进度跟踪
- 课程搜索

### Classroom 课堂
- 课程内容展示
- 视频播放
- 章节导航
- 学习笔记

### ToolsLab 工具实验室
8个专业项目管理工具：

| 工具 | 功能 | 会员等级 |
|------|------|----------|
| 蒙特卡洛模拟器 | 风险量化分析 | Pro |
| 敏捷估算扑克 | 团队协作估算 | Pro |
| Kanban流动指标 | 效率可视化 | Pro |
| 学习曲线模型 | 工期优化 | Pro |
| 挣值趋势预测 | EVM分析 | Pro+ |
| 关键链法调度 | CPM计算 | Pro+ |
| 鱼骨图分析 | 根因分析 | Pro+ |
| 质量成本模型 | COQ分析 | Pro+ |

### Simulation 实战模拟
- 场景列表
- 场景详情
- 决策演练
- AI评估报告

### Profile 个人资料
- 学习贡献热力图
- 能力雷达图
- 徽章收藏馆
- 荣誉证书墙

### KnowledgeGraph 知识图谱
- 3D节点可视化
- 学习路径规划
- AI学习助手
- 节点详情

## 后台管理页面

### AdminDashboard 管理仪表盘
- 用户统计
- 课程数据
- 活跃度分析

### UserTable 用户管理
- 用户列表
- 权限管理
- 会员管理

### AdminContent 内容管理
- 体系课程
- 核心算法
- 实战项目
- 知识图谱

### AdminProgress 学习进度
- 用户学习情况
- 课程完成率
- 学习时长统计

### AdminCommunity 社区管理
- 帖子管理
- 评论审核
- 用户互动

### AdminAnnouncements 公告管理
- 发布公告
- 历史公告
- 公告置顶

### AdminMembership 会员管理
- 会员列表
- 订单管理
- 退款处理

### AdminSimulation 模拟管理
- 场景管理
- 决策配置
- 评分规则
