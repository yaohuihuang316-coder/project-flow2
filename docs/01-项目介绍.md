# ProjectFlow - 项目管理学习平台

## 项目概述

ProjectFlow 是一个面向项目管理专业人士的在线学习平台，提供系统化的课程体系、实战模拟演练和专业工具支持。

## 核心功能

### 1. 学习中心
- **体系课程**: 从基础到专家级的完整项目管理课程
- **实战模拟**: 10+ 真实项目场景演练
- **专业工具**: 8个项目管理实用工具

### 2. 会员体系
| 等级 | 权益 | 解锁条件 |
|------|------|----------|
| Free | 基础课程、3个工具 | 注册即可 |
| Pro | 全部课程、5个专业工具 | 完成5门课程 |
| Pro+ | 全部功能、AI报告、模拟中心 | 完成10门课程 |

### 3. 个人成长
- **学习贡献热力图**: 记录每日学习活动
- **能力雷达图**: 六维能力评估（规划/执行/预算/风险/领导/敏捷）
- **徽章系统**: 13个成就徽章等你解锁

## 技术栈

- **前端**: React 18 + TypeScript + Vite + Tailwind CSS
- **后端**: Supabase (PostgreSQL + Auth + Storage)
- **部署**: Vercel

## 快速开始

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建部署
npm run build
```

## 环境变量

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
VITE_MOONSHOT_API_KEY=your_moonshot_key
```
