## 场景一: 新用户首次体验 (5分钟)

### 步骤 1: 登录与 Dashboard 总览
**操作:**
1. 访问 `http://localhost:5173`
2. 使用 Free 用户登录: `free@test.com` / `test123`
3. 观察 Dashboard 加载动画

**解说词:**
> "登录后首先看到个人仪表盘，这里聚合了所有学习数据。顶部显示连续学习天数，激励用户保持学习习惯。"

**检查点:**
- [ ] 用户名正确显示
- [ ] Streak 天数显示
- [ ] 趋势图正常渲染
- [ ] 知识图谱节点浮动动画

### 步骤 2: 知识图谱交互
**操作:**
1. 在 Dashboard 找到知识图谱模块
2. 鼠标悬停查看节点动画
3. 点击 "PMBOK" 节点

**预期结果:**
- 自动跳转到 Classroom 页面
- 显示对应课程内容

**解说词:**
> "知识图谱是我们的特色功能，每个节点对应一个课程，点击即可开始学习。"

- [ ] 进度条更新

**操作:**
1. 返回 Dashboard

## 场景七: 管理后台 (8分钟)

2. 使用管理员账号登录: `admin@test.com`
3. 自动进入 Admin Dashboard

### 步骤 18: 公告管理
**操作:**
1. 进入"全站公告"
2. 点击"新建公告"
3. 输入标题: "系统维护通知"

5. 设置目标受众: "全部用户"
6. 设置优先级: "高"
7. 保存并发布

**解说词:**
> "管理员可发布定向公告，支持按会员等级筛选目标受众。"

**验证:**
1. 切换回普通用户账号
2. 查看导航栏公告红点
3. 点击展开公告弹窗

### 步骤 19: 会员管理
**操作:**
1. 进入"会员管理"
2. 查看用户列表
3. 演示调整用户等级
4. 进入"兑换码"标签
5. 点击"生成兑换码"
6. 设置: Pro+ 等级，数量 5，永久有效
7. 生成并复制

**解说词:**
> "后台支持批量生成兑换码，用于企业培训和营销活动。"

**操作:**
1. 进入"模拟场景"

   - 阶段 1: "资源冲突发现"

5. 保存并发布

## 🎉 演示结束

### 总结话术
> "ProjectFlow 是一个完整的项目管理学习平台，涵盖课程学习、实战模拟、社区互动和会员体系。系统采用 React + Supabase 架构，支持实时数据同步和权限分级。"

### 技术亮点回顾
1. **响应式设计** - 适配各种屏幕
2. **权限控制** - 4级会员体系
3. **实时同步** - Supabase Realtime
4. **数据可视化** - Recharts + 自定义图谱
5. **AI 集成** - Gemini API
