## 🆘 故障排查

### 常见问题

**问题1: 页面白屏**
```
解决: 检查浏览器控制台报错
      确认 Supabase 配置正确
```

**问题2: 数据不加载**
```
解决: 检查网络连接
      确认 RLS 策略配置
      检查表中有数据
```

**问题3: AI 无响应**
```
解决: 检查 API Key
      查看调用次数限制
```
